# A.M Braids - Website Profissional de Tranças

## 🎨 Sobre o Projeto

Website moderno, responsivo e dinâmico para **A.M Braids Trancista** - especialista em tranças profissionais com mais de 10 anos de experiência.

## ✨ Características

### Design & UX
- **Paleta de Cores Premium**: Rosa Magenta (#E7057A), Amarelo (#FFFF00), Rosa Claro (#FFB6D9), Cinza (#777777)
- **Responsivo**: Funciona perfeitamente em mobile, tablet e desktop
- **Menu Hamburger**: Navegação móvel otimizada com animações suaves
- **Animações Dinâmicas**: Transições ao scroll, hover effects elegantes, blobs animados

### Seções
1. **Header** - Logo e navegação responsiva com menu mobile
2. **Hero** - Landing impressionante com chamadas à ação
3. **Sobre** - Apresentação de Agatha com foto profissional e estatísticas
4. **Portfolio** - Galeria dinâmica com 14 fotos de trabalhos realizados
5. **Contato** - Links para Instagram, Email, WhatsApp e Localização
6. **Footer** - Rodapé com informações e links rápidos

## 🛠️ Tecnologias

- **Next.js 16** com App Router
- **React 19** com Hooks modernos
- **TypeScript** para segurança de tipos
- **Tailwind CSS v4** para styling responsivo
- **Lucide Icons** para ícones profissionais
- **Next.js Image Optimization** para performance

## 📁 Estrutura de Componentes

```
components/
├── header.tsx       # Navegação com menu responsivo
├── hero.tsx         # Seção inicial com animações
├── about.tsx        # Apresentação de Agatha
├── portfolio.tsx    # Galeria de trabalhos
├── contact.tsx      # Seção de contato com links sociais
└── footer.tsx       # Rodapé

app/
├── layout.tsx       # Layout principal (pt-BR)
├── page.tsx         # Página inicial
└── globals.css      # Estilos globais e animações
```

## 🎯 Funcionalidades Dinâmicas

- **Scroll Suave**: Navegação fluida entre seções
- **Intersection Observer**: Animações ativadas ao entrar na viewport
- **Menu Mobile**: Abre/fecha com efeito de fade
- **Hover Effects**: Cards e botões com transições elegantes
- **Lazy Loading**: Imagens otimizadas com Next.js Image

## 🚀 Performance

- Imagens otimizadas via Vercel Blob Storage
- CSS animado com performance otimizada
- Sem dependências externas pesadas
- Carregamento rápido com code splitting automático do Next.js

## 📱 Responsividade

- **Mobile First**: Otimizado para telas pequenas
- **Breakpoints**: sm (640px), md (768px), lg (1024px)
- **Menu Hamburger**: Ativado automaticamente em telas < 768px
- **Grid Adaptativo**: Layouts que se ajustam por tela

## 🔧 Configuração Local

```bash
# Instalar dependências
pnpm install

# Rodar em desenvolvimento
pnpm dev

# Build para produção
pnpm build

# Iniciar produção
pnpm start
```
